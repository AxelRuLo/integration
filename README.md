### ToolsetAI - Backend

1. Clone the repository

2. Install dependencies
```bash
npm install
```

3. Create .env file and fill the env variables with the .env.template_example file

4. Start DB 
```bash
docker-compose up -d
```

5. Run the app in dev mode
```bash
npm run start:dev
```
